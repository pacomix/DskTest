3.1.6 ISO C2y preview

Use --std-c2y to compile in this mode.

Implementation status

In anticipation of the upcoming version of the language standard, C2y, SDCC supports the following prospective features:

  • N3260: _Generic selection expression with a type operand
  • N3370: Case range expressions
