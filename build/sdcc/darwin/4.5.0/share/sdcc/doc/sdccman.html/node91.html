3.8.6 SM83 and TLCS-90 Interrupt Service Routines

SDCC does not attempt to generate an interrupt vector table.

Type Syntax
Behaviour
Interruptible interrupt handler void f(void) __interrupt
Interrupt handler can be interrupted by further interrupts
Non-interruptible interrupt handler void f(void) __critical __interrupt(0)
Interrupt handler cannot be interrupted by further interrupts
