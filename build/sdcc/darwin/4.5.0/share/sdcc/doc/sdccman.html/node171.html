4.8.1 Startup Code

On the MOS6502 the startup code is inserted by linking with crt0.rel which is generated from sdcc/device/lib/mos6502/crt0.s. If you need a different startup code you can use the compiler option --no-std-crt0 and provide your own crt0.rel. When using a custom crt0.rel it needs to be listed first when linking.
